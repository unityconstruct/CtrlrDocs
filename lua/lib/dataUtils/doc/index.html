dec2bin (decNum)
converts a decimal number to its binary analog. Uses a for loop to count down from 7 to 0 Index is used for both the table index & the power of 2: 2^i

Parameters:

  • decNum integer number 0 to 255

Returns:

    string binary representation of the decimal number
